# RTS Partners — Relatórios BTG

## Como usar o relatório de Posições BTG

### O que você precisa
- Computador com internet
- Um navegador (Chrome, Edge, Firefox)
- Seu login RTS Partners (e-mail e senha)

### Passo a passo

1. Abra o arquivo `btg_positions.html` com duplo clique (abre no navegador)
2. Digite seu **e-mail** e **senha** RTS Partners
3. Clique em **Entrar**
4. Selecione a **data** desejada
5. Use os filtros de **Banker** e **Código Cliente** se quiser filtrar
6. Clique em **Buscar**
7. Para exportar os dados para Excel, clique em **↓ Exportar CSV**

### Filtros disponíveis

| Filtro | Descrição |
|--------|-----------|
| Data | Data de referência das posições |
| Banker | Filtra por banker responsável |
| Código Cliente | Filtra por cliente específico (ex: DN-AT) |

### Abrir o CSV no Excel

Ao exportar, abra o arquivo `.csv` no Excel. Se os dados aparecerem em uma só coluna:
1. Selecione a coluna A
2. Menu **Dados** → **Texto para Colunas**
3. Escolha **Delimitado** → **Ponto e vírgula**
4. Clique em Concluir
